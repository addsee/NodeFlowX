# ✨ نبذة عن المشروع

**NodeFlowX** هو بيئة تطوير مرئية متقدمة (Visual Scripting Environment) صُممت لتبسيط عملية تطوير الألعاب عبر نظام العُقد (Nodes)، حيث يمكن للمطور إنشاء المنطق البرمجي بصريًا دون الحاجة إلى كتابة كل شيء يدويًا.

يهدف المشروع إلى توفير تجربة احترافية تجمع بين:

- ⚡ سهولة الاستخدام
- 🚀 الأداء العالي
- 🎨 التصميم الحديث
- 🌍 دعم تعدد المحركات
- 🧠 تحويل المخططات البصرية إلى أكواد حقيقية قابلة للتشغيل

---

# 🎮 المحركات المدعومة

| المحرك | نوع الكود |
|---|---|
| 🎮 Unity | C# |
| 🌀 Godot | GDScript |

---

# 🚀 المميزات (Features)

## 🎨 تجربة مستخدم حديثة ومتطورة

- 🧩 نظام سحب وإفلات سلس للنودات
- 🔗 توصيلات ذكية بين العقد
- 🔍 مساحة عمل تدعم التكبير والتحريك (Pan & Zoom)
- 🗺 خريطة مصغرة (Mini-map) للمشاريع الكبيرة
- 💡 شروحات ذكية (Smart Tooltips)
- 🌙 واجهة حديثة ومتجاوبة
- 🖥 دعم الويب وسطح المكتب

---

## 🌍 دعم اللغات والثيمات

- 🇸🇦 دعم كامل للعربية والإنجليزية
- 🔄 تبديل فوري بين اللغات
- 🌗 الوضع الليلي والنهاري
- 💾 حفظ التفضيلات تلقائيًا باستخدام GetStorage

---

# 🔄 توليد الأكواد (Forward Generation)

يقوم NodeFlowX بتحويل النودات والمخططات البصرية إلى أكواد حقيقية قابلة للتشغيل داخل محركات الألعاب الحديثة.

### المحركات المدعومة حاليًا:

| المحرك | اللغة |
|---|---|
| Unity | C# |
| Godot | GDScript |

---

# 🧠 التحويل من الكود إلى نودات (Reverse Engineering)

ميزة متقدمة تسمح بتحليل الأكواد وتحويلها إلى مخططات مرئية قابلة للتعديل.

### الصيغ المدعومة

- C#
- GDScript

> ⚠️ هذه الميزة ما تزال تجريبية وقد لا تدعم جميع التركيبات البرمجية.

---

# 🛠 أدوات إضافية

- 👁 نافذة استعراض الأكواد
- 📋 نسخ الكود بضغطة واحدة
- 🧩 نظام إدارة المتغيرات
- 🌐 متغيرات محلية وعالمية
- 💾 حفظ تلقائي للمشاريع
- 🔄 دعم تبديل المحركات

# 🧱 الهيكلة البرمجية (Architecture)

تم بناء المشروع باستخدام **GetX** لضمان الأداء العالي وسهولة الصيانة والتوسع.

```txt
lib/
├── controllers/
├── services/
├── models/
├── views/
├── widgets/
└── core/
```

---

## 📂 المكونات الرئيسية

| المجلد | الوظيفة |
|---|---|
| controllers | إدارة حالة المحرر والنودات |
| services | توليد الأكواد وتحليلها |
| models | نماذج البيانات |
| views | الشاشات الرئيسية |
| widgets | مكونات الواجهة القابلة لإعادة الاستخدام |
